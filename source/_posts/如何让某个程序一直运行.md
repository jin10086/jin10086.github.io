因为有时候需要让某段程序一直运行,停止了就自动重启...

从网上找了好久，找到了这段代码。。。用了挺好，记录下

```python
def main():
    print("AutoRes is starting")

    # 你要一直运行的代码放在这...
    executable = sys.executable
    args = sys.argv[:]
    print(args)
    args.insert(0, sys.executable)

    time.sleep(1)
    print("Respawning")
    os.execvp(executable, args)


if __name__ == "__main__":
    main()
```
